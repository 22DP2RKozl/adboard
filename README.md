# AdBoard

## Instrukcija kā palaist programmu
1. Ielādēt `XAMPP`
2. Pārkopēt `adboard` mapi "../xampp/htdocs/" mapē
3. Atvērt `XAMPP` un palaist `Apache` un `MySQL` servisus
4. Atvērt http://localhost/phpmyadmin/index.php interneta pārlūkā
5. PHPMyAdmin vietnē `SQL` sadaļā ievietot un palaist "ad_board.sql" faila saturu
6. Atvērt http://localhost/adboard/index.php interneta pārlūkā

## Izmantotās tehnoloģijas
- PHP
- CSS
- JavaScript
- MySQL
